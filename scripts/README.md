# Utility Scripts

This directory contains utility scripts generated by the AI assistant to perform various maintenance and migration tasks for the Living on Earth website.

## Scripts

### `scrape_archives.py`
A Python script used to scrape content from the legacy `loe.org` website. It handles:
- Fetching show and segment data.
- extracting metadata (title, date, summary, images).
- Converting HTML content to Markdown.
- Saving content in the Pelican content structure.

### `convert_logo.py`
A Python script using Pillow to convert the legacy JPEG logo (`logo_square_v2.jpg`) into a PNG with a transparent background (`logo_square_v2.png`).
